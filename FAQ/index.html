Tạo tài khoản Github

Git là tên gọi của một Hệ thống quản lý phiên bản phân tán (Distributed Version Control System – DVCS) là một trong những hệ thống quản lý phiên bản phân tán phổ biến nhất hiện nay.

Github chính là một dịch vụ máy chủ repository công cộng, mỗi người có thể tạo tài khoản trên đó để tạo ra các kho chứa của riêng mình để có thể làm việc

Github là nơi mình có thể chạy các website tĩnh và là nơi lưu trữ các dự án, các source code của mình.

Bước 1: Tạo tài khoản Github

vào trang chủ của Github để đăng ký một tài khoản miễn phí

Bước 2: Tạo mới một repository trên Github

Sau khi đăng ký xong tài khoản chúng ta tạo một repository(gọi tắt là repo là kho chứa dữ liệu, source của bạn)

Tiếp tục làm theo hướng dẫn

Sau khi tạo xong một repo(repository) bạn đã có một nơi để lưu trữ các đoạn mã nguồn. Tại đây bạn có thể chỉnh sửa, đẩy code ở máy tính của mình lên github cũng như có thể tải toàn bộ kho chứa về máy tính của mình để làm việc

Bước 3: Tải github-desktop

Tải Github Desktop đây là công cụ giúp bạn đẩy code lên Github cũng như tải các repo từ Github về máy tính. Tùy theo hệ điều hành bạn chọn tải phiên bản cho Window/Mac.Bạn có thể sử dụng command-line/terminal để làm các thao tác tương tự.

Sau khi tải và cài đặt thành công, bạn đăng nhập bằng tài khoản Github, giao diện của công cụ như sau

Bước 4: clone một repository từ Github

Tiếp tục click vào Clone a Repository để tải repository của bạn từ Github về máy tính của mình. Chọn repo muốn tải, chọn nơi lưu trữ và click clone như hình dưới:

Bước 6: Lấy đường dẫn của file index.html

Khi tạo một repository có dạng useNameGithub.github.io chúng ta có thể chạy được các trang web tĩnh.

click vào file index.html tiếp đó click Copy Patch sau đó dán vào url của trình duyệt ta đã chạy được một trang web tĩnh với domain dạng như sau https://useName.github.io/project
